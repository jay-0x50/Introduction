구현 내용

핵심 시스템

  • 노트 낙하 시스템: 일정한 속도로 화면 상단에서 하단으로 원형 판정 영역까지 낙하하는 애니메이션
  • 판정 로직: 4단계 판정 (Perfect/Good/Bad/Miss) - 노트 도착 시각과 키 입력 시각의 오차 범위로 계산
  • 점수 계산: 판정 레벨별로 다른 가중치 적용, 콤보 시스템
  • 키 입력 처리: 이벤트 큐 방식으로 빠른 입력 감지, 데드존 관리

기술적 학습

  • 게임 루프와 델타 타이밍 (deltaTime을 이용한 프레임 독립적 이동)
  • 상태 머신 패턴 (메뉴 → 게임 중 → 리절트)
  • 사운드 동기화 (BGM 재생 시간과 노트 타이밍 맞추기)

/ PROJECT 07

Voice-Controlled Web Game

기술: JavaScript, Web Speech API
개발: 개인 프로젝트
규모: HTML/CSS/JS 약 1,500줄

프로젝트 개요

마이크로 입력받은 음성 명령으로 게임을 조작합니다. "마법 주문" 음성 인식을 통해 캐릭터의 스킬을 발동하는 재미있는 웹 기반 인터랙티브 게임입니다.

기술 특징

  • Web Speech API: 브라우저 내장 음성 인식 기능 활용
  • DOM 조작: 음성 명령에 따른 CSS 애니메이션과 클래스 변경
  • 실시간 처리: 음성 인식 결과를 즉시 게임 상태에 반영
