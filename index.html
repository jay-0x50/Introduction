지원 동기
"플레이의 재미를 구현으로 연결하는 게임플레이 프로그래머"

저는 어릴 때부터 단순히 배우는 데서 그치지 않고, 직접 만들어 보고 움직이게 하는 과정에서 더 큰 흥미를 느끼는 사람이었습니다. 초등학교 3학년 때 스크래치로 처음 코딩을 접했고, 이후 관련 자격증도 취득하며 자연스럽게 프로그래밍에 관심을 키워 갔습니다. 중학교에서는 SW 동아리에서 파이썬을 배우며 흥미를 더 키웠고, C언어까지 배우며 기초를 넓혀 갔습니다.

특히 중학교 2학년 때는 오렌지보드를 활용해 자동 손 소독제 기계를 직접 구현해 학급에 기부한 경험이 있습니다. 이 경험을 통해 저는 단순히 코드를 작성하는 것을 넘어, 내가 만든 결과물이 실제로 쓰이고 도움이 되는 과정에서 큰 보람을 느끼는 사람이라는 점을 깨달았습니다. 이런 경험들이 쌓이며 저의 관심은 자연스럽게 사람들에게 가장 직접적으로 재미와 경험을 전달할 수 있는 게임 개발로 이어졌습니다.

어린 시절부터 저는 게임을 즐길 때마다 "이 통쾌한 타격감은 어떤 수식으로 만들어졌을까?", "이 절묘한 판정은 어떤 규칙으로 돌아갈까?"라는 호기심을 품어왔습니다. 저에게 코딩은 단순히 기능을 완성하는 작업이 아니라, 플레이어가 손으로 느끼는 조작감과 화면에서 확인하는 피드백을 자연스럽게 연결하는 일입니다. 보이지 않는 로직을 안정적으로 설계해 기획 의도가 플레이 경험으로 정확히 전달되도록 만드는 게임플레이 프로그래머가 되고자 지원합니다.

프로젝트
경험
"장르를 넘나들며 다진 플레이 로직 구현 경험"

학창 시절 동안 2,300시간 이상, 10여 개의 프로젝트를 진행하며 플레이어가 직접 체감하는 코어 로직을 꾸준히 구현해 왔습니다. 2D 리듬 게임 'PyMax'에서는 노트 스폰, 입력 처리, 사운드 동기화, 판정 시스템을 직접 설계하며 리듬 게임에서 가장 중요한 입력 감각과 피드백의 일치에 집중했습니다.

'SpaceOut' 프로젝트에서는 DirectX 11 기반으로 렌더링과 충돌 처리 로직을 직접 구현했습니다. 이 경험을 통해 엔진 기능을 사용하는 데 그치지 않고, 게임 규칙이 한 프레임 안에서 어떤 순서와 구조로 동작해야 하는지 이해할 수 있었습니다. 덕분에 장르와 도구가 달라도 플레이 로직을 빠르게 구조화하는 힘을 길렀습니다.

팀 프로젝트인 '마법소녀카와이러블리즈'에서는 6인 팀의 메인 프로그래머로서 보스 레이드 코어 로직과 서버 연동 기능을 맡아 협업 환경에서의 책임감을 배웠습니다. 또 'JSoooooot'과 'JacMan'에서는 교내 사용자 대상 배포 이후 피드백을 수집해 실제 플레이 반응에 맞춰 코드를 수정하며, 구현 이후의 개선 과정까지 경험했습니다.

자기소개서 박재영
성격의
장점
"맡은 시스템을 끝까지 책임지는 협업 방식"

팀 프로젝트에서 가장 기본적이면서도 중요한 협업의 원칙은 '내가 맡은 역할을 끝까지 책임지고 완수하는 것'이라고 굳게 믿고 있습니다. 저는 전공 프로젝트 개발 동아리 'CodeUs'의 부장으로서, 그리고 다수의 팀 프로젝트에서 메인 프로그래머를 맡아오며 게임의 핵심 로직과 시스템 구현을 안정적으로 마무리하는 자세를 지켜 왔습니다.

저에게 할당된 업무를 안정적이고 신속하게 마무리한 후에는 늘 팀원들의 개발 상황을 살피며, 문제가 발생했을 때 주저 없이 디버깅과 트러블슈팅을 도왔습니다. 교내 학우들을 대상으로 배포하여 직접 피드백을 수렴하고 코드 리뷰를 진행했던 경험처럼, 예상치 못한 오류 앞에서도 좌절하지 않고 팀과 유저의 빈틈을 찾아 메우는 조력자가 되겠습니다.

입사 후
포부
"플레이어 경험을 중심에 두는 개발자"

프로그래머로서 저의 궁극적인 목표는 기획자가 의도한 참신함과 창의성을 어떠한 기술적 제약 없이 화면 위에 온전히 구현해 내는 것입니다. 이를 위해 저는 개발 과정 내내 끊임없이 '어떻게 하면 더 나은 조작감과 재미를 줄 수 있을까?'를 치열하게 고민하며 작성한 코드와 시스템 구조를 지속적으로 다듬고 리팩토링합니다.

지금까지 다양한 장르의 메인 로직을 직접 구현하며 쌓아 온 경험을 바탕으로, 입사 후에도 팀의 개발 방식과 프로젝트 방향을 빠르게 익히겠습니다. 안정적인 코드와 확장 가능한 구조를 바탕으로 플레이어가 반복해서 손이 가는 재미를 만드는 데 기여하는 개발자로 성장하겠습니다.
